git diff: mostra diferenças entre os commits de um anterior para o novo

git restore index.html: restaura um arquivo especifico

git restore .: restaura todos os arquivos modificados

isso quando estiver apenas no feito o git add .

quando os arquivos ja foram comitados chama-se de stage

para desfazer alterações ja comitadas comandos abaixo

git restore --stage restaura todos os arquivos para o estagio anterior que é apenas de git add .

se eu quiser restaurar tudo mesmo dou o comando git restore . ou nome do arquivo especifico

corrigindo o ultimo Commit: git commit --amend -m "change title page index.html"

Como desfazer todos os Commits: git reset --soft HEAD~1
